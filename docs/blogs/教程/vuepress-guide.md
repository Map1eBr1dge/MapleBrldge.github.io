# VuePress 博客搭建指南

本文记录了使用 VuePress v2 + vuepress-theme-reco + GitHub Pages + GitHub Actions 搭建个人博客的完整过程。

## 技术选型

| 技术 | 用途 |
|------|------|
| VuePress v2 | 静态网站生成器 |
| vuepress-theme-reco | 博客主题 |
| GitHub Pages | 免费托管 |
| GitHub Actions | 自动部署 |

## 搭建步骤

1. 初始化项目并安装 VuePress
2. 配置 vuepress-theme-reco 主题
3. 编写 Markdown 文章
4. 配置 GitHub Actions 自动部署
5. 在 GitHub Pages 中启用部署

## 心得体会

搭建博客的过程本身就是一个很好的学习机会，从 Node.js 包管理到 CI/CD 流程，每一步都有收获。
