# Hello World

欢迎来到 MapleBridge 的个人博客！

## 关于这个博客

这个博客使用 VuePress v2 + vuepress-theme-reco + GitHub Pages + GitHub Actions 搭建。

## 技术栈

- **VuePress v2**: 静态网站生成器
- **vuepress-theme-reco**: 美观的博客主题
- **GitHub Pages**: 免费静态网站托管
- **GitHub Actions**: 自动化部署

## 主题特性

- 🎨 精美的博客主题
- 📱 响应式设计，支持移动端
- 🔍 内置搜索功能
- 💬 评论系统支持
- 📊 文章分类和标签
- ⏱️ 时间轴展示
- 🚀 返回顶部按钮
